Hey, I'm Daniel. Check out my personal game projects below, and check Github for more projects!

Voxilent Alpha

The very first project. Starting as a simple clicker this project rapidly evolved into a large-scale factory building game. My introduction to the concept of scope-creep.

  • Learn More

Super Pixel Riders

Taking the lessons learned from Voxilent Alpha, Super Pixel Riders is a top down car-combat game. Unlock guns and vehicles to use against your pursuers.

  • Learn More

Crate Pusher

Who doesn't love a puzzle game?! In Crate Pusher you have to get the Crates to their goals. Each Crate is subject to it's own unique movement rules.

  • Learn More

Breakout

A duo project undertaken with a friend who wanted to learn more about game development. It's breakout. That's it.

  • Learn More

Civ Helper

One of my favourite board game; Civilization: A New Dawn. Wanting to skip the admin/setup required to play a game, I created this tool to assist players with their games.

  • Learn More

Genetics

Can 'blind' rockets navigate a maze? Absolutely not. But with the power of 'procreation', a genetic lineage can find a way.

Gallery

This is a Gallery element. It can behave as a lightbox (when given the lightbox class), and you can customize its appearance with a number of modifiers, as well as assign it an optional onload or onscroll transition modifier (details).
